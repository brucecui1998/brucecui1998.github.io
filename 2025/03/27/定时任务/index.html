要在 Linux 上设置定时任务来每天自动执行 sudo systemctl restart cpolar 命令，可以通过 cron 来实现。因为 systemctl 命令需要 sudo 权限，所以你需要确保 cron 可以在无需手动输入密码的情况下执行此命令。
步骤：

编辑 sudoers 文件以免输入密码： 你需要允许 cron 以无密码的方式执行 systemctl 命令。使用以下命令编辑 sudoers 文件：

sudo visudo

在文件中添加如下行（假设用户是 your_username）：

your_username ALL=(ALL) NOPASSWD: /bin/systemctl restart cpolar

这行配置允许 your_username 用户在执行 systemctl restart cpolar 时不需要输入密码。

创建 cron 任务： 使用 crontab 命令设置定时任务：

crontab -e

在打开的文件中，添加如下行来每天执行 systemctl restart cpolar 命令：

0 0 * * * sudo systemctl restart cpolar

这行表示：
    0 0 * * *：每天的午夜 00:00 执行任务。
    sudo systemctl restart cpolar：执行命令。

保存并退出： 如果你是使用 visudo 编辑 sudoers 文件，编辑完成后保存并退出（按下 Ctrl+X，然后按 Y 确认保存，最后按 Enter 退出）。在 crontab 文件中编辑完成后，保存并退出编辑器。

验证：

你可以通过以下命令来查看当前的 cron 任务是否已成功添加：

crontab -l

这会列出所有定时任务，确认你的任务已正确添加。

这样设置后，系统将每天自动在午夜执行 sudo systemctl restart cpolar 命令，重启 cpolar 服务。
