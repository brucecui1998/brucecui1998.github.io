1351. 统计有序矩阵中的负数

题目

给你一个 m * n 的矩阵 grid，矩阵中的元素无论是按行还是按列，都以非递增顺序排列。 请你统计并返回 grid 中 负数 的数目。

示例 1：

示例 2：

提示：

  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 100
  • -100 <= grid[i][j] <= 100

进阶：你可以设计一个时间复杂度为 O(n + m) 的解决方案吗？

思路

二分查找：

所以我们通过二分查找可以找到每一行中从前往后的第一个负数，那么这个位置之后到这一行的末尾里所有的数必然是负数了，可以直接统计

复杂度分析

时间复杂度：二分查找一行的时间复杂度为logm，需要遍历n行，所以总时间复杂度是O(nlogm)
空间复杂度：O(1)

倒序遍历：

考虑我们已经算出第 i 行的从前往后第一个负数的位置 posi，那么它下面的那个元素肯定也是负数。我们可以在遍历每一行时从后往前遍历，一旦遇到正数就break，遇到负数就记录它的位置，下一次接着从这个位置遍历

时间复杂度：考虑每次循环变量的起始位置是单调不降的，所以起始位置最多移动 每一行有多少列 次，时间复杂度 O(n+m)。
空间复杂度：O(1)
