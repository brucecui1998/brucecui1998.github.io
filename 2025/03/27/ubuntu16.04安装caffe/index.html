配置ssh

  1. 安装 OpenSSH 服务器

使用 apt 包管理器来安装 OpenSSH 服务器：

sudo apt update
sudo apt install openssh-server

安装完成后，OpenSSH 服务会自动启动并设置为开机自启。你可以通过以下命令检查 SSH 服务 是否正在运行：

sudo systemctl status ssh

如果 SSH 服务正在运行，你应该看到类似以下的输出：

ssh.service - OpenBSD Secure Shell server
Loaded: loaded (/lib/systemd/system/ssh.service; enabled)
Active: active (running) since Fri 2025-03-20 10:00:00 UTC; 5min ago

  1. 配置防火墙以允许 SSH 连接（22端口）

如果你启用了 UFW (Uncomplicated Firewall) 防火墙，你需要允许 22 端口（SSH 默认端口）的流量。

检查 UFW 防火墙状态：

sudo ufw status

如果输出显示防火墙是 inactive，则说明防火墙没有启用。如果显示为 active，则说明防火墙已启用。

打开 22 端口：

使用以下命令允许通过 22 端口 进行 SSH 连接：

sudo ufw allow 22/tcp

这将允许 TCP 端口 22 的流量。

启用 UFW 防火墙（如果尚未启用）：

如果 UFW 防火墙尚未启用，你可以通过以下命令启用它：

sudo ufw enable

重新检查防火墙状态：

运行以下命令以确保 22 端口 已经打开：

sudo ufw status

你应该看到类似如下的输出，表示 22 端口 已允许通过：

Status: active

To                         Action      From
--                         ------      ----
22/tcp                     ALLOW       Anywhere
  1. 测试 SSH 连接

现在，SSH 服务已经安装并且防火墙配置好了。你可以使用以下命令在同一局域网或远程计算机上测试 SSH 连接：

ssh username@

其中，username 是你要连接的用户名， 是你 Ubuntu 16.04 服务器的 IP 地址。

安装 NVIDIA 驱动的步骤

步骤 1: 禁用 nouveau 驱动

编辑 blacklist 文件：

使用文本编辑器打开 /etc/modprobe.d/blacklist.conf 文件：

sudo nano /etc/modprobe.d/blacklist.conf

添加禁用 nouveau 驱动的条目：

在文件的末尾添加以下两行：

blacklist nouveau
options nouveau modeset=0

更新 initramfs：

禁用 nouveau 后，需要更新初始化内存文件系统：

sudo update-initramfs -u

重启计算机：

重启计算机以使禁用 nouveau 生效：

sudo reboot

GCC版本

ubuntu16.04内核编译的gcc版本是gcc 5.4.0

你在安装 NVIDIA 驱动时遇到的错误 Warning: Compiler version check failed 和 Building kernel modules 主要是因为用于编译 NVIDIA 驱动的编译器与内核编译时所使用的编译器版本不匹配。内核和驱动需要使用相同版本的编译器来确保正确编译模块。
错误原因：

内核和编译器版本不匹配：内核使用的是 gcc 5.4.0，但是 NVIDIA 驱动的安装脚本正在尝试使用 gcc 5.3.1 进行编译，导致了版本不匹配的问题。

解决方法：

  1. 安装内核头文件和相关开发工具

确保你已经安装了正确版本的内核头文件以及构建所需的开发工具。这些文件用于编译驱动。

sudo apt update
sudo apt install build-essential linux-headers-$(uname -r)

linux-headers-$(uname -r)：这将安装与你当前运行的内核版本相对应的内核头文件。
build-essential：这个包包含了 GCC 编译器和其他必要的开发工具。
  1. 安装与内核匹配的编译器版本

确保你正在使用正确的 GCC 版本。如果内核是用 GCC 5.4.0 编译的，你应该确保安装 GCC 5.4.0 版本。

检查当前 GCC 版本：

gcc –version

如果不是 5.4.0，你可以尝试安装该版本。

安装 GCC 5.4.0：

如果你没有安装 GCC 5.4.0，执行以下命令来安装：

sudo apt install gcc-5 g++-5

切换 GCC 版本：

然后通过以下命令设置 gcc 和 g++ 为版本 5.4.0：

sudo update-alternatives –install /usr/bin/gcc gcc /usr/bin/gcc-5 50
sudo update-alternatives –install /usr/bin/g++ g++ /usr/bin/g++-5 50
